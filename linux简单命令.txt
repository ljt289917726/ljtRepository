cd命令

用途：改变当前工作目录。

示例：cd /home/username 切换到用户的家目录。

ls

用途：列出目录内容。

示例：ls 列出当前目录的内容。

ll 或 ls -l

用途：以长格式列出目录内容，显示详细信息（如权限、所有者、大小、修改日期等）。

示例：ls -l 列出当前目录内容的详细信息。

mkdir

用途：创建新目录。

示例：mkdir newdir 创建一个名为newdir的新目录。
mkdir -p mkdir/mwdir

rm
  
用途：删除文件或目录。       rm -rf 强制删除

示例：rm file.txt 删除名为file.txt的文件。rm -r dir 删除名为dir的目录及其内容。

cp

用途：复制文件或目录。

示例：cp source.txt dest.txt 将source.txt复制为dest.txt。

touch

用途：创建一个空文件或更新文件的访问和修改时间。

示例：touch newfile.txt 创建一个名为newfile.txt的空文件。

pwd

用途：显示当前工作目录的完整路径。

示例：pwd 显示当前目录的路径。

mv

用途：移动或重命名文件或目录。

示例：mv oldname.txt newname.txt 重命名文件。mv file.txt /path/to/destination/ 将文件移动到指定路径。

vi 或 vim

用途：文本编辑器，用于创建和修改文本文件。

示例：vi filename.txt 打开或创建名为filename.txt的文件进行编辑。i进入编辑模式 esc退出编辑模式进入命令模式， 在vi中，常用的命令包括:w保存，:q退出，:wq保存并退出等。

常用的tar命令

(1)压缩和解压.tar.gz文件

打包.tar.gz文件：tar -czvf mysql-5.7.tar.gz mysql

解压.tar.gz文件到当前目录：tar -xzvf mysql-5.7.tar.gz

解压.tar.gz文件到指定目录：tar -xzvf mysql-5.7.tar.gz -C /home/apps/software/

2、如何查看自己的防火墙属于 iptables 还是 firewalld
运行命令：sudo firewall-cmd --state
iptables防火墙,则会出现内容：
iptables: Firewall is not running.
iptables: Firewall is running.   
firewalld 防火墙,则会出现内容：
running
2、永久关闭防火墙(通过命令)：
1、对于 iptables 防火墙：
sudo service iptables stop
sudo chkconfig iptables off
2、对于 firewalld 防火墙：
sudo systemctl stop firewalld
sudo systemctl disable firewalld
